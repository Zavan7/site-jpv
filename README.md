# Site Institucional - Igreja Jesus o Pão da Vida (JPV)

Este projeto consiste no desenvolvimento de um **site institucional para a Igreja Evangélica Pentecostal Jesus o Pão da Vida**.

O repositório foi criado com o objetivo de **praticar desenvolvimento web e organização de projetos utilizando Django**, além de aplicar conceitos de **HTML, CSS e JavaScript** na construção de uma interface moderna e responsiva.

⚠️ **Status do projeto:**

Alterações na estrutura, funcionalidades e layout podem ocorrer frequentemente.

# Arquitetura Utilizada (MVT)

O projeto utiliza o padrão arquitetural **MVT (Model – View – Template)**, que é o padrão adotado pelo Django.

De forma simplificada:

**Model**  
Responsável pela estrutura dos dados e pela comunicação com o banco de dados.

**View**  
Contém a lógica da aplicação. Recebe requisições do usuário, processa os dados e retorna uma resposta.

**Template**  
Responsável pela camada de apresentação. São os arquivos HTML que exibem as informações para o usuário.

Fluxo simplificado:

```
Usuário → View → Model → View → Template → Usuário
```

Esse padrão ajuda a manter o projeto **organizado, modular e de fácil manutenção**.
